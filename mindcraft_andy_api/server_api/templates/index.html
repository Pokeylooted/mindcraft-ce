Mindcraft Andy API

The API is up and running.

This is a placeholder page. Future developments might include API status, documentation links, or a list of active compute providers.

Endpoints:

  • POST /generations/join_pool: Allows compute providers to join the pool.
  • POST /completions: Accepts prompts for text generation.
  • GET /api/models: Lists models available in the community pool.
  • GET /debug/providers: (Debug) Shows current compute providers.
