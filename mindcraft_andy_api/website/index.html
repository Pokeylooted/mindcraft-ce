How It Works

Mindcraft Community AI connects users seeking AI completions with individuals and groups generously sharing their compute resources. Our intelligent routing ensures your requests are handled efficiently.

Want to contribute? You can run our client application to add your local Ollama models to the pool and help power the community!

API Access

To use the models, send a POST request to https://mindcraft.riqvip.dev/completions with a JSON payload like:


{
  "model": "model_name:tag", // e.g., "llama3:8b"
  "prompt": "Your text prompt here...",
  "stream": false // Optional, true for streaming
}

The model_name:tag should be one of the models listed above. If a specific tag (e.g., :8b-q4_K_M) is not available, the system will try to match the base model name (e.g., llama3).
